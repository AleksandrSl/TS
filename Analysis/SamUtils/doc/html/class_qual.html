Detailed Description

An iterable container class for the Qual field of a SAM/BAM. Requires the BAMRead that the Qual is retrieved from to still remain in scope.

QUAL: ASCII of base QUALity plus 33 (same as the quality string in the Sanger FASTQ format). A base quality is the phred-scaled base error probability which equals 10 log10 Prfbase is wrongg. This field can be a `*' when quality is not stored. If not a `*', SEQ must not be a `*' and the length of the quality string ought to equal the length of SEQ
