Detailed Description

A class to allow the user to iterate through the SAM/BAM file/stdin. A file must already be opened by the reader for an iterator to be valid.

example code

string bam_file = string(argv[1]); BAMReader reader(bam_file); for (BAMReader::iterator i = reader.get_iterator(); i.good(); i.next()) { BAMRead read = i.get(); cerr << read.to_string(); }
