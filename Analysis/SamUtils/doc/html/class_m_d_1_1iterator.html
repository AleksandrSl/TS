Detailed Description

MD::iterator is how you access the actual elements of hte MD string. Currently only a forward iterator.

example code. Assumes a valid BAMRead object named "read" MD md = read.get_md(); for(MD::iterator md_iterator = md.get_iterator(); md_iterator.good(); md_iterator.next()) { cout << md_iterator.get(); }
