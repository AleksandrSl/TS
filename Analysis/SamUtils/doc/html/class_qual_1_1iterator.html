Detailed Description

Qual::iterator is how you access the actual elements of hte Qual string. Currently only a forward iterator.

example code. Assumes a valid BAMRead object named "read" Qual qual = read.get_qual(); for(Qual::iterator qual_iterator = qual.get_iterator(); qual_iterator.good(); qual_iterator.next()) { cout << qual_iterator.get(); }
