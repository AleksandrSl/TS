Detailed Description

the PileupFactory::pileup_iterator class abstracts all the positional information from the user of the class It also removes the possibility of dealing with positions that have no reads overlapping them. This will iterate only over full Pileup objects (it's possible that the Pileup has no coverage however).

Use this for maximum efficiency when looking at the entire window.

WARNING!!! The iterator is not guaranteed to iterate the contents of the Factory in order.

example code assuming properly constructed object. Assume an existing factory named pileups

for (PileupFactory::pileup_iterator itr = pileups.get_pileup_iterator(); itr.good(); itr.next()) { Pileup& p = itr.get(); cerr <<delim<< "Ref id:"<<delim<< p.get_tid() <<delim<< "position in genome:"<<delim<< p.get_pos() <<delim<<"num reads:"<<delim<< p.get_num_reads() << endl; }
